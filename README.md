# rhizome
The API that feeds grass roots movements.

## Prerequisites ##
You'll need gulp:
`npm install -g gulp && npm install gulp`
You'll need nodemon:
`npm install -g nodemon`

Then you'll need to grab the latest modules:
`npm install`

## Building ##
`npm run build`
`npm start`; or
`npm run dev`
